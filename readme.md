## 制作アプリのタイトル   今日何食うたと？. 

## 制作アプリの説明（40文字程度）. 
- 食品名を入力するとカロリーと三大栄養素が検索できる.
- 場面別・1日の合計を表示.  
- カロリーの消費するための運動を表示. 
## 工夫した点・こだわった点. 
- 海外のAPIだったため当初英語検索しかできず、Google翻訳APIも追加した点. 
- 登録する際にMySQLで一意のIDを付与して登録した食品を選択→１件削除を実装
- 検索ボタンクリック後に非同期処理で「検索中・・・」を表示. 
- カレンダー実装. 
- グラム入力で栄養素を計算し表示. 
- 摂取したカロリーを消費するための運動をAPIを使って表現. 

## 次回トライしたこと（または機能）. 
- さくらのデータベースとPHPを接続して開発.
- 食品ごとの消費カロリーを取得、表示. 
- APIではなく栄養素のデータベースを作ってみる.
