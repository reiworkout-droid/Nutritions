## 制作アプリのタイトル   何をどれだけ食うてんねん. 

## 制作アプリの説明（40文字程度）. 
食品名を入力するとカロリーと三大栄養素が検索できる。.
場面別のボタンクリックでテキストエリアに保存、場面別・1日の合計を表示.  

## 工夫した点・こだわった点. 
- 海外のAPIだったため当初英語検索しかできなかったため、Google翻訳APIも追加した点. 
- 登録した食品に一意のIDを付与して選択→１件削除を実装
- オブジェクト内の栄養素別数値を表示用と別に計算用として保存し、合計値を算出、表示. 
- 検索ボタンクリック後に非同期処理で「検索中・・・」を表示. 

## 次回トライしたこと（または機能）. 
- カレンダー表示、クリックしてから入力画面に遷移、選択した日をオブジェクトに入れて保存管理.
- グラム入力で栄養素を計算し表示. 
- 摂取したカロリーを消費するための運動をAPIを使って表現.
